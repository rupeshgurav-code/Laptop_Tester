# Laptop Tester

Laptop Tester is a full-stack diagnostic application that helps users test the essential hardware components of their laptops, including the keyboard, webcam, battery, audio, and microphone. After running the tests, the system generates a detailed report of the results.

## 🚀 Tech Stack
- **Frontend:** React.js
- **Backend:** Express.js (Node.js)
- **Database:** MySQL (via XAMPP)
- **Version Control:** Git & GitHub

## ✨ Features
- Test laptop hardware components:
  - Keyboard
  - Webcam
  - Battery
  - Audio
  - Microphone
- Generate a diagnostic report
- Store results in MySQL database
- Clean separation of frontend and backend code
